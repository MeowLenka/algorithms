1. Нельзя сделать int &ref = 1; 
потому что ссылка должна быть привязана к l-value (левому значению), 
а 1 - это r-value (правое значение).

L-value - объект, который имеет имя и адрес в памяти (переменные)
R-value - временное значение, которое не имеет имени и адреса 
(литералы, результаты выражений)
2. Перегрузка функций в C++ - это создание нескольких функций 
с одинаковым именем, но разными параметрами.


_VA_ARGS__ — это специальный макрос в языке C, 
который используется для работы с переменным числом 
аргументов в макросах.

6. рекурсия
recur_summ(4)
│
├── 4 + recur_summ(3)
     │
     ├── 3 + recur_summ(2)
          │
          ├── 2 + recur_summ(1)
               │
               └── return 1  ← Базовый случай!
               │
               return 2 + 1 = 3
          │
          return 3 + 3 = 6
     │
     return 4 + 6 = 10
